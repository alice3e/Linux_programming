Компиляция и линковка программы - это два отдельных этапа процесса создания исполняемого файла.

Компиляция - это процесс преобразования исходного кода на языке программирования в объектные файлы, которые содержат машинный код и символьную информацию. Компиляцию можно выполнить с помощью команды g++ -c:


g++ -O3 -c sha1_benchmark.cc sha1.cc


Эта команда скомпилирует исходные файлы sha1_benchmark.cc и sha1.cc в объектные файлы sha1_benchmark.o и sha1.o соответственно.

Линковка - это процесс объединения объектных файлов в исполняемый файл. Линковку можно выполнить с помощью команды g++ -o:


g++ -O3 sha1_benchmark.o sha1.o -o sha1-with-opt


Эта команда выполнит линковку объектных файлов sha1_benchmark.o и sha1.o в исполняемый файл sha1-with-opt.

После успешной компиляции и линковки, вы можете запустить программу, используя команду:


./sha1-with-opt
