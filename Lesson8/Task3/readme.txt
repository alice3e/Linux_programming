
    Создайте директорию для сборки:

bash

meson build

    Перейдите в директорию сборки:

bash

cd build

    Запустите процесс сборки:

bash

meson compile

Теперь у вас должен быть исполняемый файл с именем myprogram в директории build. Meson автоматически управит зависимостями, включая zlib, если она установлена на вашей системе.
