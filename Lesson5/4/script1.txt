Для создания скрипта, который будет читать сообщения из именованного канала и выводить их на экран вместе с текущей датой, вы можете использовать следующий пример скрипта на языке Bash:

```bash
#!/bin/bash

# Путь к именованному каналу
fifo_path="/tmp/my_named_pipe"

# Создаем именованный канал, если он не существует
if [ ! -p "$fifo_path" ]; then
    mkfifo "$fifo_path"
fi

# Бесконечный цикл для чтения и вывода сообщений
while true; do
    if read -r line < "$fifo_path"; then
        # Получаем текущую дату и время
        current_date=$(date "+%Y-%m-%d %H:%M:%S")
        
        # Выводим сообщение с датой
        echo "[$current_date] $line"
    fi
done
```

Сохраните этот скрипт, например, как `named_pipe_reader.sh`, и дайте ему права на выполнение:

```bash
chmod +x named_pipe_reader.sh
```

Теперь, чтобы продемонстрировать работу скрипта, запустите его в одном окне терминала:

```bash
./named_pipe_reader.sh
```

В другом окне терминала выведите какие-нибудь строки в именованный канал `/tmp/my_named_pipe`, например, с помощью команды `echo`:

```bash
echo "Привет, мир!" > /tmp/my_named_pipe
echo "Это сообщение будет отображено в другом окне терминала." > /tmp/my_named_pipe
```

Сообщения будут автоматически читаться скриптом и выводиться в окне, где был запущен скрипт `named_pipe_reader.sh`, с указанием текущей даты и времени.
